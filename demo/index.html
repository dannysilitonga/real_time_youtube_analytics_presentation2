Real Time Sentiment Video Analytics: Intro

This project will utilize both backed and front ends. The former is designed for data ingestion, data processing and data modeling on real-time basis, while the later for data visualization.

Real Time Sentiment Video Analytics: Diagram

Real Time Sentiment Video Analytics: Backend

The backend processing works as follows:

  • The backend processing entails connecting to Youtube API
  • After the data is extracted, it's cleaned and transformed
  • The data is fed to a sentiment analysis model

Real Time Sentiment Video Analytics: Frontend

The frontend processing works as follows:

  • Connect to Kaggle Dataset and extract the data
  • Create a visualization of top trending videos over time
  • Create a visualization of sentiments by categories and how it has changed over time
